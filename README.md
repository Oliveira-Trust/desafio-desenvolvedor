### Funcionalidades🚀
- Autenticação
- Criar usuário
- Fazer cotação em Dólar, Libra e Euro
- Possibilidade de receber uma cópia da cotação por email
- Sincronização automática de preços
- Possibilidade de alterar as taxas (É necessário entrar com o admin)
