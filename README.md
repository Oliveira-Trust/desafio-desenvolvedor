## Instruções de Instalação

Aplicar os seguintes comandos:

- composer install
- Configurar o Laravel (banco no .env e tal)
- npm install
- npm run prod
- composer dumpautoload
- php artisan db:seed --class=DatabaseSeeder  (para alimentar o banco com alguns dados)
