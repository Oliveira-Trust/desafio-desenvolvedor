## Sobre o projeto

CRUD's para gestão de clientes, produtos e pedidos.

- Crud de clientes. (Usando o model User)
- Crud de produtos.
- Crud de pedidos.

## Banco de dados

Uso de MySQL com as tabelas sendo criadas pelo sistema de migrations do Laravel.
